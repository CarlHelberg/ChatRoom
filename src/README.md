# Simple Online Chat Application

## Overview
For CS1103, week7 from UoPeople.

This is a simple online chat application built using Java socket programming. 
It allows multiple clients to connect to a central server, send messages, and receive messages from other clients in real-time.

### Features:
- Multi-client support
- Unique user IDs assigned to each client
- Message broadcasting to all connected clients (Chatroom)
- Text-based/CLI user interface

## Running the Application

### Requirements:
- Java 8 or higher
